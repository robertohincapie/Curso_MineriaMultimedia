# Unidad 1 — Fundamentos de Deep Learning para Datos No Estructurados

## Idea central
Comprender cómo entrenar modelos neuronales modernos para procesar datos no estructurados.

## Pregunta guía
¿Cómo aprenden las redes neuronales a partir de ejemplos?

## Objetivos
- Comprender la representación mediante tensores.
- Implementar modelos en PyTorch.
- Construir pipelines completos de entrenamiento.
- Diagnosticar problemas de generalización.

## Temas

### 1. Introducción a PyTorch
- Tensores
- Operaciones matriciales
- CPU y GPU
- Autograd

### 2. Redes neuronales básicas
- Neurona artificial
- Capas lineales
- Redes secuenciales
- Representaciones internas

### 3. Funciones de activación
- Sigmoid
- Tanh
- ReLU
- Leaky ReLU
- Softmax

### 4. Aprendizaje supervisado
- Funciones de pérdida
- MSE
- Cross Entropy
- Optimización

### 5. Descenso de gradiente
- Forward pass
- Backpropagation
- Actualización de pesos

### 6. Gestión de datos
- Dataset
- DataLoader
- Batching
- Shuffle
- Validación

### 7. Generalización
- Underfitting
- Overfitting
- Early Stopping
- Dropout
- Weight Decay

### 8. Pipeline completo
Datos → Dataset → DataLoader → Modelo → Loss → Optimizer → Entrenamiento → Evaluación → Inferencia

Clasificación de imágenes simples utilizando PyTorch.
